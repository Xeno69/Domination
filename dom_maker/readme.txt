Needs MakePBO and DePbo from mikero https://mikero.bytex.digital/Downloads
需要 MakePBO 跟 DePbo 来自 mikero https://mikero.bytex.digital/Downloads

Just copy MakePBO.exe and DePbo.dll in the dom_maker folder, edit make_dom.bat and change the version number twice, save, run make_dom.bat
只需复制 MakePBO.exe 跟 DePbo.dll 到 dom_maker 文件夹，编辑 make_dom.bat 并再次更改版本号，保存，运行 make_dom.bat

After a while you have all versions in the dom_maker folder.
一段时间后，所有版本都在 dom_maker 文件夹中

Please note the source folder for all missions is always co30_Domination.Altis, no matter which version
请注意，所有任务的源文件夹始终为 co30_Domination.Altis ，无论哪个版本
